常用命令

容器生命周期管理

run

根据一个镜像创建一个新容器，不填加选项则会自动停止

语法：docker run [OPTIONS] IMAGE [COMMAND] [ARG...]

OPTIONS说明：

  • –name 为容器指定一个名称
  • -d 后台方式运行
  • -it 使用交互方式运行，进入容器中查看内容
  • -p 指定容器的端口
    • -p 主机端口：容器端口 将主机端口映射给容器端口

实例：

使用docker镜像nginx:latest以后台模式启动一个容器,并将容器命名为yyflightNginx：

运行centos容器，并进去系统中

stop、start、restart

停止、启动、重启多个已经纯在的容器，可以使用名称，也可以使用容器id

kill

杀死一个容器，比起stop，使用kill命令会导致信息未保存从而丢失

容器操作

ps

  • 列出当前运行的容器，如果没有容器运行，则什么都不显示
  • -a 显示所有运行的容器，并显示历史开启的容器
  • -q 只显示容器的编号

查看本地正在运行的容器

退出容器交互：

exit： 退出容器，并关闭容器

ctrl + p + q ： 容器不停止退出

top

查看指定容器中运行进程情况

inspect

查看镜像的源数据

exec

进入已经开启的容器中

cp

将容器中的文件拷贝到宿主机
