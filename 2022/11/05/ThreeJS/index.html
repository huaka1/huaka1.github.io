加载.gltf文件

注意：

  • load函数的第一个参数url，因为我们最开始目录是index.html，所以应该 ./src/components/three/scene/model.gltf

  • 加载glb文件和gltf文件没有什么区别

  • gltf格式模型可以包含平行光光源DirectionalLight、点光源PointLight、聚光源SpotLight。

  • gltf模型不能包含three.js中的环境光光源AmbientLight、半球光光源HemisphereLight、平面光光源RectAreaLight。

纹理

orbitcontrols

实现滚轮放大缩小，移动旋转的功能

Constructor

OrbitControls( object : Camera, domElement : HTMLDOMElement )

object: （必须）将要被控制的相机，通过调整相机的角度，相对的使得物体变了。

domElement: 用于事件监听的HTML元素，只有在规定的html执行鼠标操作才能变换
